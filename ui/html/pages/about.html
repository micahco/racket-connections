About

Racket Connections is an online board for OSU students interested in playing court sports and making friends.

Software

Racket Connections is built with Go along with the following open source software:

  • PostgresSQL - database
  • pgx - pure Go driver and toolkit for PostgreSQL
  • Gomail - SMTP interface
  • chi - router
  • scs - session management
  • nosurf - CSRF protection middleware
  • Tailwind CSS - stylesheets

Hosted on DigitalOcean with Caddy. SMTP testing and delivery with Mailtrap.

Much of the source code was built while following along with Alex Edwards in his book Let's Go.

No JavaScript is required to use this site, but there are a few pages that include some scripts.
